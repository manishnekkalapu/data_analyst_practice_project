# Sales Dashboard Project

## Project Overview
This project analyzes sales data using Excel, SQL, and Power BI to identify product performance and city-wise sales trends.

## Tools Used
- Microsoft Excel
- SQL
- Power BI

## Dataset Information
The dataset contains:
- Order ID
- Product Name
- City
- Sales Amount

## Key Insights
- Milk is the highest-selling product.
- London generated higher sales compared to Manchester.
- Total sales amount reached approximately 2K.

## Visualizations Created
- Bar Chart: Sales by Product
- Pie Chart: Sales by City
- Card: Total Sales
- Slicer: City Filter

## Skills Demonstrated
- Data Cleaning
- SQL Queries
- Data Aggregation
- Dashboard Creation
- Data Visualization
